# Energy Information Exhibition App

A simple script that opens a website with information about the different exhibits in the energy exhibition. 


## Installation

Assuming you're using Linux Mint, just 
```bash
git clone https://github.com/jerusalem-science-museum/energy-information.git
cd energy-information
chmod +x start_firefox_kiosk.sh
```
then go to startup applications and add the script as a custom startup app.
